# Cadeau voor Lisette en Pieter

Een zelfstandige cadeauwebsite in de stijl van de trouwkaart, met het lied
`Weg van jou` en foto's en video's uit de opnamestudio.

## Lokaal bekijken

```powershell
python -m http.server 8766
```

Open daarna `http://127.0.0.1:8766`.

## Publiceren

De repository bevat een `render.yaml` voor een Render Static Site.
